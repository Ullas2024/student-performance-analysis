# Student Performance Analysis

This project analyzes student academic performance using Python.
The goal is to understand how factors like study time, attendance,
and previous scores affect final results.

## Features
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Data visualization using charts
- Performance insights using Python

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Use Case
Helpful for educators and institutions to understand student trends
and improve academic outcomes.
